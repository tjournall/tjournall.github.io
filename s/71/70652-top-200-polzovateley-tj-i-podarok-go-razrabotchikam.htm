Топ-200 пользователей TJ и подарок Go-разработчикам

Начнём с конца, а точнее с подарка:

6294 просмотров

Это библиотека для работы с API сайтов «Комитета».

Пока поддерживает не все методы. Список нереализованных методов лежит в файле todo.txt в корне репозитория, но они скучные и никому не нужны, поэтому мне было лень.

Начать работу с библиотекой можно тремя способами: авторизация по токену из QR-кода, использование существующего токена уже авторизованного пользователя, использование некоторых методов без авторизации вовсе.

Зачем это, например, надо?

Первый пример использования библиотеки — анализатор лайков в комментариях. Вдохновилась работой Марка Варламова (саму библиотеку тоже поэтому делать начала). Написано отвратительно, но мне в тот день хотелось спать. Работает медленно, потому что у API есть ограничение: необходимый метод можно вызывать максимум один раз в секунду. Этот пример существует только в виде исходного кода, можете компилировать и запускать. Не оформляю нормально, потому что уже есть рабочий вариант от Марка.

Второй пример — общедоступный топ-200 пользователей TJ.

Этот топ тоже написан с использованием библиотеки. Топ обновляется раз в сутки, примерно в 00:30 по Москве. Сейчас он отображает только рейтинг на момент последнего обновления и его изменение за сутки.

Главная задача этого сервиса — отображать топ шутников, то есть людей, набравших наибольшее количество плюсиков за месяц и получивших бесплатную подписку. Я обязательно добавлю на страницу такой рейтинг, когда данных будет достаточно: бесплатные подписки раздаются первого числа каждого месяца, а сервису всего несколько дней.

Кстати, не обижайтесь, что страничка свёрстана как говно. Я абсолютно не фронтенд-разработчик, поэтому просто украла какой-то шаблон. На мобильных устройствах выглядит не очень.
