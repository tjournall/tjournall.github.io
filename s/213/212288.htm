Развитый навык искать информацию и интуитивное ориентирование в поисковиках буквально расширяет ваш объем памяти ресурсами интернета.
Сегодня я расскажу о поисковых операторах, которые помогут сделать результат поиска куда более конкретным и полезным. Если вы ищете альтернативу расширенному поиску и всё ещё не слышали о операторах, тогда эта статья для вас.

Операторы поисковика Google

@ - социальные сети.

Проводит поиск по данным социальных сетей.
После содержания запроса укажите название социальной сети с приставкой в виде оператора "@".

Пример запроса: "Xbox @twitter".

"#" - хештеги.

Проводит поиск по хештегам.
Просто загуглите нужный хештег.

Пример запроса: " #животные ".

"$" - поиск цен.

Ищет цены.
Удобно использовать с оператором ".."(диапазон).
Пример запроса: "камера $400".

"filetype:" - расширение файла.

Проводит поиск документов указанного формата.
Укажите оператор "file:" c нужным форматом после запроса.
Другой вариант оператора: "ext:".

Пример поиска: "живая типографика filetype:pdf".

"related:" - похожий контент.

Ищет сайты с похожим контентом на указанный сайт.
После оператора "related:" укажите url сайта, с которым ищете сходство.

Пример запроса: "related:https://developer.mozilla.org/ru/".

"-" - минус слова.

Исключает нежелательные слова из запроса.
Сначала введите слово, по которому будете искать, после чего укажите слова, которые хотите исключить, с приставкой в виде символа "-".

Пример запроса: "мэнсон -мэрилин -marilyn".

"OR" - логическое или.

Показывает результаты, где встречается хотя бы один из желаемых результатов.
Это может быть полезно, если вы не помните точного названия и хотите перебрать несколько вариантов. Укажите возможные желаемые результаты поиска, разделив их оператором "OR".

Пример запроса: "Зимбардо OR Милгрэм".

".." - диапазон.

Проводит поиск в числовом диапазоне.
Просто укажите оператор ".." между двумя числовыми операндами.

Пример запроса: "1917..1934 + год".

"define:" - определение.

Возвращает определение термина из авторитетного источника по вашему запросу. Укажите термин после оператора "define:".

Пример запроса: "define:презумпция".

"allinurl:" - присутствие в URL.

Ищет определенные слова в составе URL сайтов.После оператора "allinurl:" укажите слова, которые хотите найти. Так же можно указать ключевое слово перед оператором, используя оператор "inurl:".

Пример запроса: "allinurl:seo".

"allintext:" - контекст страницы.

Ищет только по контексту страницы, исключая описание, url и все другое.
Укажите свой запрос после оператора "allintext:".

Можно указать ключевое слово перед оператором, используя оператор "intext:".
Для поиска по заголовку, используйте операторы "allintitle": и "intitle:"

Пример запроса: "allintext:семантика htm".

"site:" - поиск по сайту.

Очень полезный оператор.Позволяет искать информацию исключительно по конкретному сайту. После запроса, укажите оператор "site:" c названием сайта.

Пример запроса: "meta site:https://developer.mozilla.org/ru/".

' "" ' - кавычки.

Ищет точное соответствие запросу, указанному в кавычках.
Просто заключите ваш вопрос в кавычках, чтобы исключить все возможные вариации вашего ключа поиска.

Пример запроса: "Элемент <canvas> представляет удобный API для рисования".

"cache:" - кэш.

Позволяет проводить поиск в кеше.
Может быть очень полезной функцией, если вам надо найти то, что уже удалено. После оператора "cache:", укажите url сайта, в кеше которого хотите искать.

Пример запроса: "cache:edx.org".
